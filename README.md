# 📖 About The Project

VLITS CGPA Calculator is designed to help students evaluate their overall academic performance by calculating cumulative GPA across semesters.

The platform helps students:

- Track academic growth
- Calculate cumulative performance
- Monitor semester-wise progress
- Estimate final graduation CGPA
- Make informed academic decisions

This application serves as a practical academic companion throughout a student's educational journey.
